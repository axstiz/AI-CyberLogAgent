# Готовые конфигурации генератора логов

Эта директория содержит предустановленные JSON-конфигурации для различных сценариев генерации логов Apache.

## Быстрый старт

```bash
# Использование готовой конфигурации
python -m log_gen.cli --config log_gen/configs/realistic_mixed.json --output output.log
```

## Доступные конфигурации

### 1. `realistic_mixed.json` - Реалистичная смешанная
**Рекомендуется для начала**

- **Логов:** 800
- **Инциденты:** 8%
- **Ошибки:** 12%
- **Описание:** Сбалансированная конфигурация с реалистичным распределением всех типов логов. Включает как нормальную работу, так и различные типы ошибок.
- **Использование:** Общее тестирование, обучение моделей, демонстрация

```bash
python -m log_gen.cli --config log_gen/configs/realistic_mixed.json --output mixed_logs.log
```

## Модификация конфигураций

Вы можете скопировать любую конфигурацию и изменить под свои нужды:

```bash
# Копировать конфигурацию
cp log_gen/configs/realistic_mixed.json my_custom_config.json

# Отредактировать параметры
# ... редактирование JSON файла ...

# Использовать модифицированную конфигурацию
python -m log_gen.cli --config my_custom_config.json --output custom.log
```

## Параметры для настройки

Основные параметры, которые можно изменить:

- `log_count` — количество логов
- `time_increment_seconds` — интервал между логами [мин, макс]
- `incident_probability` — вероятность инцидента (0.0 - 1.0)
- `log_type_weights` — веса для каждого типа лога (сумма = 1.0)
- `client_ip_ranges` — источники IP адресов
- `forbidden_paths` — пути для ошибок доступа

Подробнее см. [CONFIGURATION.md](../CONFIGURATION.md)

## Валидация конфигурации

Перед использованием проверьте, что:

1. **Сумма весов = 1.0:**
   ```python
   sum(log_type_weights.values()) == 1.0
   ```

2. **Вероятности в диапазоне [0, 1]:**
   ```python
   0.0 <= incident_probability <= 1.0
   0.0 <= error_probability <= 1.0
   ```

3. **Диапазоны корректны:**
   ```python
   min < max для всех [min, max] параметров
   ```

## Создание собственной конфигурации

Используйте CLI для создания базовой конфигурации:

```bash
python -m log_gen.cli --create-config my_config.json
```

Это создаст файл с настройками по умолчанию и комментариями.
